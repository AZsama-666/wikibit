# GoDaddy DNS 配置指南 - www.up9.life

## 步骤说明

### 步骤1：登录GoDaddy账户

1. 访问：https://www.godaddy.com
2. 点击右上角的 "Sign In"（登录）
3. 输入你的账号和密码登录

### 步骤2：进入域名管理

1. 登录后，点击右上角的头像/账号名称
2. 在下拉菜单中选择 **"My Products"**（我的产品）
3. 或者直接访问：https://sso.godaddy.com/products
4. 找到你的域名 **up9.life**，点击 **"DNS"** 按钮

### 步骤3：添加CNAME记录（推荐方法）

1. 在DNS管理页面，找到 **"Records"**（记录）部分
2. 点击 **"Add"**（添加）按钮
3. 选择记录类型：**CNAME**
4. 填写以下信息：
   - **Name（名称）**：输入 `www`
   - **Value（值）**：输入 `azsama-666.github.io`
   - **TTL（生存时间）**：选择 `600 seconds`（或默认值）
5. 点击 **"Save"**（保存）按钮

### 步骤4：添加A记录（如果CNAME不支持，或想要根域名支持）

如果你也想让 `up9.life`（不带www）也能访问，需要添加A记录：

1. 在DNS管理页面，点击 **"Add"**（添加）按钮
2. 选择记录类型：**A**
3. 填写以下信息：
   - **Name（名称）**：输入 `@`（表示根域名）
   - **Value（值）**：输入以下IP地址之一（需要添加4条A记录）：
     - `185.199.108.153`
     - `185.199.109.153`
     - `185.199.110.153`
     - `185.199.111.153`
   - **TTL（生存时间）**：选择 `600 seconds`
4. 重复步骤3，为每个IP地址创建一条A记录（共4条）
5. 每条记录都使用相同的名称 `@`

### 常见问题

#### Q1: 找不到DNS按钮？
A: 确保你登录的是GoDaddy主账户，并且域名是在这个账户下的。如果域名已经转移到其他账户，需要登录对应的账户。

#### Q2: 已经存在www记录怎么办？
A: 如果已经有www的CNAME或A记录，你需要：
1. 找到现有的www记录
2. 点击记录右侧的编辑（铅笔图标）或删除（垃圾桶图标）
3. 删除旧记录后，添加新的CNAME记录

#### Q3: Name字段应该填什么？
- 对于www子域名：填写 `www`
- 对于根域名：填写 `@`（在GoDaddy中，@表示根域名）
- 不要填写完整的域名（不要填 `www.up9.life`，只填 `www`）

#### Q4: TTL应该设置多少？
A: 推荐使用600秒（10分钟）或默认值。更短的TTL会让DNS更改更快生效，但会增加DNS查询频率。

#### Q5: DNS记录多久生效？
A: 通常5-30分钟内生效，但最长可能需要48小时。使用 https://dnschecker.org 检查全球DNS传播状态。

#### Q6: 如何验证配置是否正确？
A: 配置完成后：
1. 等待10-30分钟
2. 访问 https://dnschecker.org
3. 输入 `www.up9.life`
4. 查看结果，应该看到解析到GitHub的IP地址（185.199.108.x等）
